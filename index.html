Votre niveau de prévention est excellent. Continuez vos efforts de vigilance."; } else if(score >= 10) { icon.innerHTML = ''; txt.innerText = "Bonne base, mais des points restent à améliorer pour garantir une sécurité totale."; } else { icon.innerHTML = ''; txt.innerText = "Attention : votre conformité est insuffisante. Le RESTEV peut vous accompagner dans vos démarches."; } } function restartQuiz() { curQ = 0; score = 0; document.getElementById('quiz-container').classList.remove('d-none'); document.getElementById('quiz-result').classList.add('d-none'); showQ(); }
